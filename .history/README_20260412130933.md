# Mentorship Evaluation System

A full-stack mentorship evaluation platform with AI-powered review summaries.

## Tech Stack

- **Backend:** Node.js, Express, MongoDB, JWT, Google Gemini AI
- **Frontend:** React, Vite, Tailwind CSS, Axios, React Router

## Project Structure

```
├── backend/
│   └── src/
│       ├── config/         # DB connection, env config
│       ├── middleware/      # JWT auth, RBAC
│       ├── models/          # Mongoose schemas (User, Student, Review)
│       ├── routes/          # API endpoints
│       ├── services/        # Gemini AI service
│       ├── scripts/         # Data cleaning & seed
│       ├── data/            # Raw + cleaned datasets
│       └── server.js        # Express entry point
├── frontend/
│   └── src/
│       ├── components/      # Reusable UI components
│       ├── context/         # Auth context provider
│       ├── pages/           # Route pages
│       ├── services/        # Axios API client
│       ├── App.jsx          # Router setup
│       └── main.jsx         # Entry point
```

## Setup

### Prerequisites

- Node.js 18+
- MongoDB running locally (or MongoDB Atlas URI)
- Google Gemini API key (for AI summaries)

### Backend

```bash
cd backend
npm install

# Create .env file (copy from .env.example)
cp .env.example .env
# Edit .env with your values:
#   MONGODB_URI=mongodb://localhost:27017/mentorship
#   JWT_SECRET=your_secret_key
#   GEMINI_API_KEY=your_gemini_api_key

# Seed database (cleans data + creates demo users)
npm run seed

# Start server
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`, backend on `http://localhost:5000`.

## API Endpoints

| Method | Endpoint                  | Auth     | Role    | Description              |
|--------|---------------------------|----------|---------|--------------------------|
| POST   | /api/auth/login           | No       | —       | Login, returns JWT       |
| GET    | /api/auth/me              | Yes      | Any     | Get current user         |
| GET    | /api/students             | Yes      | Any     | List all students        |
| GET    | /api/students/:id         | Yes      | Any     | Get student by ID        |
| POST   | /api/reviews              | Yes      | Mentor  | Submit a review          |
| GET    | /api/reviews/:studentId   | Yes      | Student | Get reviews for student  |
| POST   | /api/reviews/:id/summary  | Yes      | Mentor  | Generate AI summary      |

## Features

- **Data Cleaning:** Deduplication, date standardization, field stripping
- **JWT Auth:** Simple login-based auth with role encoding
- **RBAC:** Mentor can submit reviews; Student can view progress
- **AI Summaries:** Google Gemini generates 3 actionable bullet points from review text
- **Role-based Dashboard:** Different views for mentors and students
- **Loading & Error States:** Full UX handling throughout
